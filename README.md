# realtime-chat-app 📱
Chat em tempo real usando nodejs e socket.io

## Motivação 😎
Este app foi criado para treinar servidores websockets com nodejs.

## Tecnologias 🖥
* [NodeJs](https://nodejs.org/)
* [Socket.io](https://socket.io/)

## Como usar 🤔
### Desenvolvimento ⌨
Na pasta principal do projeto, execute:<br >
`yarn dev`
